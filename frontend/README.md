# AgroMind

AI-driven crop recommendation and soil health diagnostic system.

## Features
- Random Forest crop prediction
- Soil nutrient diagnosis
- Fertilizer recommendation
- Real-time weather integration
- FastAPI backend
- Streamlit dashboard

## Tech Stack
Python, Scikit-learn, FastAPI, Streamlit

## Run
uvicorn backend.main:app --reload
streamlit run frontend/app.py
